# NIF Video Editor

The video presentation editor is at dashboard/video-editor.html

Features:
- Text overlays with font, size, colour, position, opacity, in/out time
- Link overlays — pill button with your NIF URL baked into the video
- Logo / watermark layers
- Camera path with Catmull-Rom keyframes
- Export: MP4 (MediaRecorder), WebM, frame sequence
- Share guide for YouTube, Instagram, TikTok, Twitter, LinkedIn, Discord
